### 🔥 Featured Projects

#### 🧠 [CodeVizz AI](https://github.com/mayur-panchal-2808/codevizz-ai) — GitHub Repository Knowledge Graph & Chat
> Turn any GitHub repo into an **interactive knowledge graph**. Explore functions, classes, and dependencies visually instead of digging through files manually.

- 🔍 AST-based parsing + OpenAI embeddings stored in **PostgreSQL with pgvector**
- 💬 RAG-based chat to answer questions about any codebase
- 🔐 Secure GitHub OAuth login with encrypted access-token storage

`TypeScript` `RAG` `OpenAI` `pgvector` `OAuth`

---

#### 🤖 [Workforce AI](https://github.com/mayur-panchal-2808/workforce-ai) — Multi-Agent Orchestration Platform
> Describe a task in plain English → LLM orchestrator breaks it into sub-tasks → runs them **in parallel** across specialized agents.

- 📡 Results streamed live to a **Kanban-style UI** via Server-Sent Events
- 🛑 Human-in-the-loop support — agents pause and ask clarifying questions mid-task
- 🏠 Runs on **local LLMs via Ollama** — zero external API costs

`TypeScript` `LangGraph` `Ollama` `SSE` `Multi-Agent`
